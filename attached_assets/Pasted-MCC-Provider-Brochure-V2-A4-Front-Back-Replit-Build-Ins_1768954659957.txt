MCC Provider Brochure V2 (A4 Front/Back) — Replit Build Instructions
=======================================================================

Goal
----
Build a small Replit app that generates a PRINT-READY PDF brochure:
- A4 size
- Page 1 = FRONT
- Page 2 = BACK
- Uses your MCC logo as a BIG CENTERED HERO LOGO on the cover
- No “salesy headline” on cover (remove “Grow Your Business. Zero Advertising.”)
- Keep “Scan to apply”
- Remove phone number; use info@mycarconcierge.com
- Add: “Ideal for: Repair shops • Mobile mechanics • Detailers • Body shops • Tint • Wrap • Tires • Glass”
- Rewrite “We have 0 members today …” into a premium “Early Access / Founding Providers” framing

Recommended Tech (fastest + best layout)
----------------------------------------
Use HTML/CSS for layout and Playwright (Chromium) to print to PDF.

Project Structure
-----------------
mcc-brochure-generator/
  package.json

  /public

Step 1 — Create the Replit
--------------------------
1) Create new Replit → Template: Node.js
2) Name: mcc-brochure-generator

Step 2 — Install dependencies
-----------------------------
In Replit Shell:

  npm i express playwright mustache
  npx playwright install chromium

Step 7 — Add the Express server (preview)
-----------------------------------------
Create: /server.js

PASTE THIS:

const express = require("express");
const fs = require("fs");
const path = require("path");
const mustache = require("mustache");

const app = express();
const PORT = process.env.PORT || 3000;

app.use(express.static(path.join(__dirname, "public")));

function loadJson(p) {
  return JSON.parse(fs.readFileSync(p, "utf8"));
}

app.get("/brochure", (req, res) => {
  const dataPath = path.join(__dirname, "data", "brochure.json");
  const tplPath = path.join(__dirname, "templates", "brochure.html");

  const data = loadJson(dataPath);
  const template = fs.readFileSync(tplPath, "utf8");

  const html = mustache.render(template, data);
  res.setHeader("Content-Type", "text/html; charset=utf-8");
  res.send(html);
});

app.get("/", (req, res) => {
  res.send(
    "<h2>MCC Brochure Generator</h2><p>Preview: <a href='/brochure'>/brochure</a></p><p>Build PDF: run <code>npm run pdf</code></p>"
  );
});

app.listen(PORT, () => {
  console.log(`Server running on port ${PORT}`);
});

Step 8 — Add the PDF build script
---------------------------------
Create: /scripts/build-pdf.js

PASTE THIS:

const fs = require("fs");
const path = require("path");
const { chromium } = require("playwright");

(async () => {
  const distDir = path.join(__dirname, "..", "dist");
  if (!fs.existsSync(distDir)) fs.mkdirSync(distDir);

  const outputPath = path.join(distDir, "MCC_Provider_Brochure_V2_A4.pdf");

  // Replit usually runs the server separately (npm start). If you want this script to
  // spin up a server itself, you can add that later. For now:
  // 1) Run `npm start` in one shell
  // 2) Then run `npm run pdf` in another shell
  const url = process.env.BROCHURE_URL || "http://localhost:3000/brochure";

  const browser = await chromium.launch();
  const page = await browser.newPage();

  await page.goto(url, { waitUntil: "networkidle" });

  await page.pdf({
    path: outputPath,
    format: "A4",
    printBackground: true,
    margin: { top: "14mm", right: "14mm", bottom: "14mm", left: "14mm" }
  });

  await browser.close();
  console.log("PDF generated:", outputPath);
})();

Step 9 — Update package.json scripts
------------------------------------
In package.json, set:

{
  "name": "mcc-brochure-generator",
  "version": "1.0.0",
  "main": "server.js",
  "scripts": {
    "start": "node server.js",
    "pdf": "node scripts/build-pdf.js"
  },
  "dependencies": {
    "express": "^4.18.0",
    "mustache": "^4.2.0",
    "playwright": "^1.40.0"
  }
}

(Version numbers can differ; Replit is fine with latest.)

How to run
----------
1) Click Run (or run): npm start
2) Open preview → visit /brochure to confirm layout
3) In a second shell run: npm run pdf
4) Output appears in /dist/MCC_Provider_Brochure_V2_A4.pdf

Notes / Customization
---------------------
