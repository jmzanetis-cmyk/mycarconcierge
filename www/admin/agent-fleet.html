Agent smokes

Daily synthetic events confirm each agent's trigger → bus → orchestrator → handler → DB pipeline is healthy. A failure pages the admin via the same email path as the spend-cap alerts. Synthetic rows are tagged with a Smoke badge in each agent's activity feed so they're easy to dismiss.
