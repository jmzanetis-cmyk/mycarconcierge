These screenshots demonstrate SMS usage in the My Car Concierge application.

## SMS Use Cases:

1. **Two-Factor Authentication (2FA)** - SMS codes sent for account security
2. **Appointment Reminders** - 24-hour reminder before scheduled service
3. **Bid Notifications** - Alerts when new bids are received
4. **Status Updates** - Job progress notifications

## Opt-In/Opt-Out:

- Users provide consent during signup by entering their phone number
- SMS consent policy linked in signup forms
- Users can manage notification preferences in Settings
- STOP keyword support for opt-out
